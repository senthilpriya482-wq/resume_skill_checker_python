## Description
A beginner-friendly Python project that analyzes resume text, identifies relevant skills using basic string matching, and provides a simple resume strength evaluation.  

This project helps to quickly check which skills are present in a resume and gives feedback on overall resume strength.

---

## Features
- Detects predefined skills from resume text.
- Lists the skills found in the resume.
- Provides a simple resume strength evaluation:
  - EXCELLENT
  - GOOD
  - NEEDS IMPROVEMENT
- Suggests areas to improve if skills are missing.

- HTML
